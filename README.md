
# Marvel App (Prueba técnica para grupo Delsud)

En este sitio vas a poder ver información sobre héroes y comics, tambien guardar una lista con tus personajes favoritos.

EL SITIO CUENTA CON LOS SIGUIENTES APARTADOS:

1) Vista general de héroes y comics separados por un filtro para una mejor experiencia de usuario.

2) Vista de héroes favoritos. 

4) Barra de navegación con modo oscuro.

5) Barra de busqueda para buscar héroes o comics depende la vista activada

6) Footer con scroll hacia arriba

7) Paginado dinamico

8) Modal con detalles del héroe y comics en los que aparecio.

9) Diseño adaptable a dispositivos móviles.




## Entorno local front

Para levantar el proyecto front-end, parate en la carpeta raíz e ingresa este comando en tu terminal

```bash
  npm run dev
```


## Variables de entorno

Para levantar el proyecto en tu entorno local create una cuenta en el sitio de Marvel API: https://developer.marvel.com vas a necesitar cuatro variables de entorno

```javascript
NEXT_PUBLIC_MARVEL_PUBLIC_KEY=tu_clave_publica
NEXT_PUBLIC_MARVEL_BASE_URL=https://gateway.marvel.com/v1/public
NEXT_PUBLIC_TS=1
NEXT_PUBLIC_HASH=tu_hash_md5
```
